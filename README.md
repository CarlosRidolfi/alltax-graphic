# All Tax Graphic
## Teste técnico para a empresa All Tax

## Como rodar este projeto
### 1 - Faça o git clone em sua máquina
### 2 - Rode o comando npm install
### 3 - Rode o comando npm run dev
### 4 - Agora basta acessar http://localhost:3000/
